Installing
==========

Using datasets
--------------

To install the package, you can use the following command:

.. code-block:: bash

    pip install anemoi-datasets

Creating datasets
-----------------

.. code-block:: bash

    pip install anemoi-datasets[create]

Contributing
------------

.. code-block:: bash

    git clone ...
    cd anemoi-datasets
    pip install .[dev]
    pip install -r docs/requirements.txt

You may also have to install pandoc on MacOS:

.. code-block:: bash

    brew install pandoc
