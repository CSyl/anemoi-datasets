############
 Installing
############

****************
 Using datasets
****************

To install the package, you can use the following command:

.. code:: bash

   pip install anemoi-datasets

*******************
 Creating datasets
*******************

.. code:: bash

   pip install anemoi-datasets[create]

**************
 Contributing
**************

.. code:: bash

   git clone ...
   cd anemoi-datasets
   pip install .[dev]
